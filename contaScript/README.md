## Traccia​
Scrivi uno script bash o python che conta il numero di file eseguibili contenuti in una directory raggruppandoli in base allo shebang interpreter.​
​
Es. di output:​
​
```
$ contaScript /usr/bin​

52 #!/usr/bin/perl5.18​
47 #!/bin/sh​
44 #!/usr/bin/perl5.28​
22 #!/usr/sbin/dtrace -s
```
